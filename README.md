# CLI Task Manager

This is a simple Command Line Interface (CLI) tool built using Node.js.
It allows users to add, delete, and list tasks directly from the terminal.

---

## Features
- Add a task
- Delete a task using index
- List all tasks
- Data stored in a JSON file

---

## Requirements
- Node.js installed

---

## Project Files

cli-tool/
- assignment5.js (main CLI logic)
- data.json (stores tasks)
- README.md (project documentation)

---

## Commands

### Add a task
node assignment5.js add "Learn JavaScript"

### List tasks
node assignment5.js list

### Delete a task
node assignment5.js delete 0

---

## Output Example

0 - Learn JavaScript  
1 - Practice Node.js
